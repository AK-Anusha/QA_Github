# SmartQA Automation Framework

This project demonstrates a scalable QA Automation Framework for **UI, API, and database testing**.  
It is built using **Selenium, Java, TestNG, Playwright, REST Assured**, and integrates with **CI/CD tools** like Jenkins and GitHub Actions.

## Features
- UI automation with Selenium WebDriver and Page Object Model
- API testing with REST Assured
- Backend validations using SQL queries
- HTML/ExtentReports for test execution results
- CI/CD integration with Jenkins / GitHub Actions
- Cross-browser testing support

## How to Run
1. Clone the repository  
2. Open in IDE (Eclipse/IntelliJ)  
3. Install Maven dependencies (`mvn clean install`)  
4. Run tests via TestNG or Maven commands
